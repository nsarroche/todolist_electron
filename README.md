# Todo list app

This is a simple todo list application created with React and Electron.

### Build

To build this app simply do:
```
npm run build
```

### Try

To try this app simply do:
```
npm install && npm start
```

### Release

To build a final release of this app simply do:
- on linux:
```
npm run build-linux
```
- on Windows:
```
npm run build-win
```
